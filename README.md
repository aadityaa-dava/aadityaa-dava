## Featured Projects

### Customer Churn Prediction & Retention Analytics

**Python • Pandas • Scikit-Learn • XGBoost • Power BI**

* Built an end-to-end customer churn analytics and retention intelligence solution using 7,032 customer records from a telecommunications provider.
* Performed data cleaning, exploratory data analysis, feature engineering, customer segmentation, predictive modeling and business intelligence reporting.
* Engineered business-focused features including Customer Lifetime Value (CLV), Service Count, AutoPay indicators, High-Risk Customer flags, Customer Value Segments and Tenure Groups.
* Developed customer personas using K-Means clustering, identifying segments such as At-Risk Premium Customers, Loyal Premium Customers, Loyal Basic Customers and Low Engagement New Customers.
* Trained and evaluated Logistic Regression, Random Forest and XGBoost models, achieving **83.75% ROC-AUC** with XGBoost.
* Built customer risk scoring models that identified **1,002 high-risk customers** and quantified approximately **$2.86M in revenue at risk**.
* Conducted churn driver analysis revealing significant impacts from contract type, fiber optic internet service, payment methods and customer tenure.
* Designed an interactive three-page Power BI dashboard featuring Executive Overview, Churn Drivers Analysis and Machine Learning Predictions & Risk Scoring.

### Healthcare Claims Fraud Detection Using Machine Learning

**Python • SQL • Scikit-Learn • Power BI**

* Built an end-to-end healthcare fraud detection pipeline using Medicare claims, beneficiary, inpatient and outpatient datasets.
* Engineered 17 provider-level fraud indicators from healthcare claims and beneficiary utilization patterns.
* Performed extensive exploratory data analysis to identify reimbursement anomalies, abnormal claim volumes, beneficiary utilization patterns and provider behavior characteristics.
* Developed machine learning models using Logistic Regression and Random Forest, achieving **96.99% ROC-AUC** and **91.09% Recall**.
* Applied SMOTE to address class imbalance and improve fraud detection performance.
* Conducted provider-level risk analysis using SQL and Python to identify high-risk healthcare providers.
* Identified key fraud indicators including Length of Stay, Inpatient Claim Volume and Total Reimbursement Amounts.
* Designed interactive Power BI dashboards supporting fraud monitoring, provider investigation, reimbursement analysis and risk assessment.
* Generated actionable business recommendations to support healthcare fraud prevention and audit prioritization.

Repository:
https://github.com/aadityaa-dava/healthcare-claims-fraud-detection
